# About `become-the-brain-os`

The community front door to the AxonOS project: a free, self-contained browser
game (v0.3.3) where the player *becomes* the operating system of a
brain–computer interface — calibrating, cleaning a live 3-channel signal,
rejecting artifacts, decoding intent under a 4 ms deadline, and enforcing a
capability-based consent gate, with a live kernel log, streaks, sound, and a
shareable grade.

It is an **honest simulation**. It touches no brain and no sensor, and says so
plainly. But every mechanic mirrors how the real AxonOS works, so a two-minute
game leads naturally into the actual open-source kernel, standard, and SDKs.

Single `index.html`, no dependencies, no tracking, deployable to any static host.

Part of the AxonOS project: https://github.com/AxonOS-org
