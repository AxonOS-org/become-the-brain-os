> ⚠️ **It's a game — not a brain.** No sensor, no neural data, no backend, nothing stored off your device. It is a *simulation* that teaches how a real brain–computer-interface operating system works.

---

Signals arrive buried in noise. For sixty seconds **you are the OS**: decode the intent before the real-time deadline, **deny** apps that ask for raw neural data, **stop** the moment consent is revoked, and **reject** artifacts before they become fake thoughts. Miss a deadline or leak a thought and the run remembers it.

## ▶ Play

- **Live:** **[axonos-org.github.io/become-the-brain-os](https://axonos-org.github.io/become-the-brain-os/)** — nothing to install.
- **Offline:** download `index.html` and open it. The whole game is one file.
- **Controls:** `1`–`4` keys or tap · `Enter` to start. Works on phones.

## How it works

| You do | The real concept |
|:-------|:-----------------|
| Decode the named intent before the bar empties | Hard real-time decoding under a bounded deadline |
| **Deny** `raw.neural.signal`; **allow** `intent.class` | Apps get *derived* intent, never raw cognition |
| **Stop** when consent is revoked | Consent as a hard gate the system can't route around |
| **Reject** blinks, noise, phantom nodes | Artifact rejection before a false intent fires |

Clean, fast, leak-free rounds build a combo and a grade. **Zero leaks = a clean run.**

## What this is — and is not

It **is** an educational game about the AxonOS operating-system layer. It **is not** the AxonOS kernel, a medical device, or a clinical system, and it reads no brain. Runtime posture: one `index.html`, no backend, no analytics, no trackers, no external scripts; `localStorage` is used only for your best score, on your device.

## Part of AxonOS

This game is the front door to **AxonOS** — a real, open-source, real-time operating system for brain–computer interfaces, written in Rust and built in the open. The four-millisecond deadline, the deny-raw-allow-derived rule, the consent gate, and artifact rejection are AxonOS's actual design, turned into something you can feel.
