**Become the Brain OS: Evolver** is a one-thumb educational browser game explaining AxonOS as the operating-system layer between neural hardware and AI.

The player stabilizes a synthetic signal, releases inside a gold timing band, captures derived intent, enforces neural permissions, respects consent revocation, rejects artifacts, and keeps all progress local.

## Non-claim boundary

This is an educational simulation. It is not a medical device, not a BCI, not the AxonOS kernel, and reads no neural data.

## Privacy

No backend. No analytics. No trackers. LocalStorage only.

## Release hygiene

The repository package contains only the public build, documentation, CI workflow, validation script, installer, and deploy script. Local temp folders such as `.ci-tmp/`, `.git-tmp/`, and `.axon-v*/` are ignored and must not be committed.
