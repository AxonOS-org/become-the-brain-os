# Gameplay Spec — v12.5.1-evolver

## Goal

Make the game playable in under 10 seconds.

## Contract

1. The player holds signal.
2. The player releases in gold.
3. The player chooses a safe OS action.
4. Mistakes produce feedback, not dead ends.
5. The run ends with a local result.

## Modes

- Guided: no choice timer.
- Run: balanced timer.
- Sprint: short timer.
- Zen: slow mode.
