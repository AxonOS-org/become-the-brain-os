**Become the Brain OS: Evolver** is a one-thumb educational browser game about the operating-system layer between neural hardware and AI.

Current build:

```text
v9.3.8-evolver
```

## v9.3.8 architecture fixes

- explicit FSM lifecycle;
- real card-selection timeout;
- idempotent result persistence;
- stable result object for copy/share/retry;
- pause/resume timer accounting;
- pointer/touch/keyboard hardening;
- deterministic seeded Daily Brain;
- clean CI and validator.

## Privacy posture

No backend. No analytics. No tracking. No neural data. LocalStorage only.

## Non-claim

This is an educational simulation, not the AxonOS kernel, not a BCI, and not medical software.
