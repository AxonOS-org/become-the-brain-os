# AxonOS

Standalone experimental build.

Path:

```text
/evolver-evolution-version/
```

This folder is isolated from the root site. It does not replace the main `index.html`.

## Run locally

From repository root:

```bash
python3 -m http.server 8080
```

Open:

```text
http://localhost:8080/evolver-evolution-version/
```

## Notes

- Single `index.html`
- No external scripts
- No backend
- No analytics
- Local storage only
